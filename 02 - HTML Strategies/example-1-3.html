Favorite HTML Tags

p tag

Primarily used to display a block of text, usually in paragraph form.

Here is an example of the p-tag being used in HTML:

<p>This is a paragraph.</p>

img tag

This tag is a self-closing tag that is used to display an image in your webpage

It is generally used with the "src" attribute, which calls on the source of the image, which can be in the form of a URL or an image file.

Here is an example of the img-tag being used in HTML:

<img src="http://www.thisisasampleimageurl.com">

br tag

This tag is an empty tag with no separate closing element, which is used to create a forced line break where one would not normally exist.

Here is an example of the br-tag being used in HTML:

<p>This is a line.<br>This is a new line.</p>

Which would then display in the following manner on your webpage:

This is a line.
This is a new line
